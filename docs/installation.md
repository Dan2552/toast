Installation
============

with Bundler (Gemfile) from Rubygems:

    source 'http://rubygems.org'
    gem "toast"

from Github:

    gem "toast", :git => "https://github.com/robokopp/toast.git"

then run

    bundle
    rails generate toast init
      create  config/toast-api.rb
      create  config/toast-api
